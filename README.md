# RecurPixel Portfolio

Personal portfolio website for Manu (RecurPixel), focused on backend engineering work, open-source NuGet packages, and technical writing.

## Live URLs

- GitHub Pages: `https://recurpixel.github.io`
- Custom Domain: `https://recurpixel.io`

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Vitest

## Local Development

```bash
npm install
npm run dev
```

## Build and Test

```bash
npm run build
npm run test
```

## Main Sections

- Hero and About
- Experience
- Projects
- Learning and Professional Development
- Books and Writing
- Contact

## Deployment

Build output is generated in `dist/` and deployed via GitHub Pages.
